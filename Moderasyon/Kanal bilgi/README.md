# Kanal_bilgi.yag

<p>
	Kanalın başlık (topic) kısmı hakkında bilgi verir.<br>
	Genellikle başlık kısmına yazılan kanal kuralları yada bot kullanım komutları öğrenmek için kullanılır.
</p>
